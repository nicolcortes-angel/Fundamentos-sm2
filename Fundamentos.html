Ques es el html

Lenguaje de marcado que estructura el contenido de una pagina we. Basado en etiquetas que pueden ser sinteticas o no semánticas

No es un lenguaje de programacion

Semantica en HTML

Etiquetas basicas

Etiqueta Descripcion
h1, h2, h3, h4, h5, h6 Titulos y subtitulos en HTML
p Parrafos
img Para mostrar las imagenes (src, alt)
ul, ol Etiquetas para definir listas ordenadas y desordenadas
Div Divisores de contenido - dar estilos
a Es redirecionar ana ruta o un link fuera o dentro de nuestra pagina
